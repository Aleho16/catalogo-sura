Diseño en CSS

Aprende a darle estilo a tus páginas web con CSS: colores, tipografías, animaciones y diseños modernos para una mejor experiencia visual.

Contenido del curso

  • 1️⃣ Introducción al lenguaje CSS
  • 2️⃣ Selectores, propiedades y valores
  • 3️⃣ Colores, fuentes y espaciado
  • 4️⃣ Diseño responsivo con Flexbox y Grid
  • 5️⃣ Animaciones, transiciones y efectos visuales

Instructor

Mariana Ortiz

Diseñadora web y Frontend Developer con más de 6 años de experiencia creando interfaces modernas y accesibles.

Requisitos

Se recomienda tener conocimientos básicos de HTML para aprovechar al máximo el curso.
