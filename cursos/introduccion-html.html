Introducción a HTML

Aprende la base de toda la web: cómo estructurar páginas con HTML y crear tus primeros sitios desde cero.

Contenido del curso

  • 1️⃣ ¿Qué es HTML y cómo funciona?
  • 2️⃣ Estructura básica de una página web
  • 3️⃣ Etiquetas esenciales (headings, párrafos, enlaces, imágenes)
  • 4️⃣ Listas, tablas y formularios
  • 5️⃣ Buenas prácticas y semántica

Instructor

Laura Gómez

Desarrolladora Frontend con 5 años de experiencia en HTML, CSS y accesibilidad web.

No necesitas conocimientos previos. Solo curiosidad por aprender y un navegador web.
